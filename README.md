# ARTS Manager

Google Apps Script(WebApp)で開発している会社向け業務アプリ。

## 目的

会社の業務を一つのアプリに集約し、管理・報告・実績確認を効率化する。

## 将来的に統合する機能

- KPI管理
- 日報
- 週報
- 月報
- 経費
- シフト
- スタッフ管理
- チャット
- AIアシスタント

## 現在の状態

β版開発中。

## 開発体制

- PM：ChatGPT
- 実装：Claude Code
- テスト・UI確認：池野

## 開発ルール

- 1タスク＝1目的
- 1バグ＝1修正
- 原因調査してから修正
- 勝手なリファクタリング禁止
- GitHub commit禁止
- clasp push禁止
- deploy禁止
- β版完成までは新機能追加禁止
- 品質優先
